文明城市应知应会

为进一步宣传普及文明城市创建知识，定州市创城办舆论宣传组决定在定州发布、定州日报、定州广播电视台、定州政府微网、定州论坛、定州房产网、定州人定州事儿、定州百姓网、定州印象、定州身边事、定州经纪人、定州全攻略、创业定州等全市30余家新媒体平台开设【河北创城记——文明城市应知应会】专栏，对全国文明城市应知应会的知识进行分期推送，进一步扩大宣传教育覆盖面，提升宣传教育效果，推动全社会加快形成学习文明知识、践行文明礼仪、投身文明创建的浓厚氛围。

文明城市创建行为规范明白纸

为规范市民行为，市创城办按照文明城市创建要求，制定了《定州市交通行为“六严禁”》《 定州市公园广场“八不准”》《 定州市市场行为“四禁止”》《 定州市就餐行为“五不准”》 ，请广大市民自觉按规范要求，文明出行、文明购物、文明出游、文明就餐，争做定州文明人，争创全国文明城。

一、定州市交通行为“六严禁”

1、严禁乱闯红灯 超越黄线； 2、严禁横穿马路 翻越横栏； 3、严禁随地吐痰 乱扔杂物; 4、严禁逆向行驶 抢道行驶； 5、严禁酒后驾驶 追逐飙车； 6、严禁乱停乱放 私占车位。

二、定州市市场行为“四禁止”

1、禁止乱扔杂物 乱倒污水； 2、禁止随地吐痰 随意吸烟； 3、禁止占道经营 私搭乱建； 4、禁止拥挤争抢 车辆乱放。

三、定州市公园广场“八不准”

1、不准乱扔烟蒂； 2、不准乱扔垃圾； 3、不准随地吐痰； 4、不准宠物便溺； 5、不准踩踏草坪； 6、不准损坏公物； 7、不准乱贴乱画； 8、不准设点赌博。

四、定州市就餐行为“五不准”

1、不准剩饭剩菜； 2、不准吸烟酗酒； 3、不准吵闹喧哗； 4、不准随地吐痰； 5、不准乱丢垃圾。
